# SevakAI - Product and Platform Roadmap

> Status: Strategic outline. This is a sequencing tool, not a delivery promise.

## Purpose

This roadmap turns the SevakAI blueprint into a staged execution plan that supports safe evolution from hackathon MVP to national-scale disaster response infrastructure.

## Phase 0 - Blueprint and Foundations

Focus:
- repository scaffolding
- architecture alignment
- interface contracts
- local environment bootstrap

## Phase 1 - MVP Disaster Operations

Focus:
- authentication
- needs capture
- volunteer registration
- offline queueing
- basic CouchDB replication
- WhatsApp and SMS ingestion
- coordinator dashboard essentials

## Phase 2 - Production Hardening

Focus:
- role hardening and audit trails
- conflict resolution workflows
- observability stack
- provider failover
- deployment automation

## Phase 3 - Intelligence Expansion

Focus:
- richer Gemini workflows
- evaluation harness maturity
- geospatial enrichment
- multilingual improvements

## Phase 4 - Regional and National Scale

Focus:
- multi-region deployments
- stronger tenancy boundaries
- burst automation
- inter-agency data exchange
